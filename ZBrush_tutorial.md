### ZBurushチュートリアル
操作
> 右クリック+Alt 視点操作。Shiftで大まかに操作できる。
> ctrlを押しながら白いマニピュレーターを動かすと選択部分を複製できる

> 左右対称解除
>> トランスフォーム/シンメトリを有効をプルダウンすることで操作できる。

サブツール
> photoshopでいうところのレイヤーに当たる
>> サブツールの結合や 分離が簡単にできる

> 挿入、アペンド
>> 新しい形を違うサブツールとして挿入できる。

> 抽出
>> マスクした個所を指定の厚みに盛り上げる
確定を押すとそれを別のサブツールとして確定できる
>> 厚みの部分をべつのポリグループとして制作できるため、それをしたい場合はこの方法が有効(例えば服などを作る場合など)

ジオメトリ
> ダイナメッシュを再適用
>> 崩れたポリゴンの再計算や、解像度を操作できる
>>> ジオメトリ/ダイナメッシュ/ダイナメッシュで適用できる。また、
何もない場所でctrl+ドラッグで再適用できる。

> トポロジー編集
>> ミラーと結合で対称部分に同型オブジェクトを複製できる

ブラシの変更
> b
>> ブラシ名を入力するとファジーで選択項目を絞り込める

ブラシ名　
> standard
>> 左クリックした場所を盛り上げる。altを押しながら操作するとくぼませることができる

> Domstandard
>> standardよりシャープな表現が可能

> move
>>　選択した周囲を含めてドラッグした個所を盛り上げられるaltで反転

> movetopological
>> moveブラシよりも細かい作業に向いている

> pinch
>> 撫でたところを中心に寄せることができる

> claybuild
>> 撫でたところを盛り上げる

> polymdynamic
>> 撫でたところを平らにすることができる

> smooth
>> shiftキーを押している時だけ適用されるブラシ
撫でた場所の形状をなだらかにする。

> mask
>> ctrlを押している時だけ適用されるブラシ
書いた個所をマスクする。マスク状態だとブラシの効果を受けなくなる。
>> ctrlドラッグでマスクを解除

ポリグループ
> shift+fで表示
>> ctrl+クリックで選択、何もないところをctrl+クリックすると選択を反転
>> ctrl+wでマスクを記憶できる。ポリグループを表示することでそれを見ることができる
>> ctrl+shift+クリックで選択箇所以外を非表示にできる。
>> ctrl+shift+ドラッグで選択箇所以外を表示できる。

ギスモの移動
> 南京錠のマークを解除し操作する(altプルダウン中)
>> ping　重心に移動する
>> 家　中心に移動する
>> 回転　ワールドの開店に追従する

インサートマルチメッシュ
> IMM primitivesで異なるオブジェクトを挿入し、それを結合する
>> オブジェクトを追加後、ダイナメッシュを更新(ctrl+ドラッグ)することで結合できる。

オブジェクトが複製された場合 (Shift+s)
> 新規ドキュメントから、何もない場所をドラッグして解決する。

保存方法
> ZPR : undo情報も含まれた保存形式。ファイルから保存
> ZTL : サブツール情報だけ保存形式。ツール/ ツール保存から保存。ツール読み込みからこの形式を読むことでロードできる。
