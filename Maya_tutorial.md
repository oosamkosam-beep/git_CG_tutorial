### Mayaチュートリアル
設定
> ディスプレイ/ポリゴン/Buckfacecullingで裏面選択をしないようにできる

ピボットの移動
> ピボットを移動できる(オブジェクトの原点)

マニピュレーター
> q : 選択
> w : 移動
> e : 回転
> r : 拡大縮小
>> ctrl + shift + 右ドラッグでマニピュレータの方向をオブジェクト依存(local)かワールド依存か変更することができる

> shift+ドラッグで複数選択、
>> 選択箇所を解除したくないときはctrl+shift+ドラッグ


オブジェクトのペアリング
> 複数オブジェクトを選択して、pをプルダウンすると、最後に選択したオブジェクトが親に、他をそのオブジェクトの子にできる。
> アウトライナから選択項目を中ドラッグで他オブジェクト上でカーソルを話すと、そのオブジェクトの子にすることができる


マテリアルの適用
> ウィンドウ/レンダリングエディタ/ハイパーシェード

マテリアルにテクスチャを適用する
> 作ったマテリアルをハイパーシェードからプルダウン、カラーの横のチェッカーをプルダウン、アトリビュートエディタから、テクスチャファイルのパスを指定する。

UVの調整
> ウィンドウ/モデリングエディタ/UVエディタ

チャネルボックスエディタ
> ディスプレイ
>> v : 表示非表示
>> r : メッシュが触れるか、触れないか

モデリングツールキット　
> 接続
>> blenderのループカットのこと

> ターゲット連結ツール
>> くっつけたい頂点をドラッグすると頂点が結合す

> 四角ポリゴン描画
>> 主にリトポロジで使う
>> 点を選択し、shiftクリックで四点に囲まれた箇所に面を張る。
>> **ctrl**でループカット(**ctrl+shift**でループを削除できる)
>> **shift**ドラッグで撫でた個所をなだらかにできる
>> **tabドラッグ**で撫でたと場所に面を張れる。
>>> 面の大きさはクワッドストリプの幅の値で変更できる
>> **tab+辺を選択しドラッグ**することで辺を押し出せる
>>> 延長の項目をエッジからループにすればループからすべて伸ばせる
>> #注意
>> 上の磁石マークをクリックし、ライブサーフェイスがONである必要がある
>> 自動で全選択(ソフト選択)がONであるため、**b**を押して無効化すると、頂点ごとに微調整ができる
>>> 四角ポリゴン描画は新規オブジェクトとして作成される。そのため作業を再開したい場合は、新しくできたオブジェクトを選択し、モデリングツールキットから四角ポリゴン描画を選択する必要がある
>>> **3キー**を確認することができる

ライティング
> **Arnold/lights**で好きなライトを入れられる

レンダリング
> 設定は**ウィンドウ/レンダリングの設定/プリファレンス/レンダー設定**から
>> **共通/ファイル出力/イメージフォーマット**出力画像形式を選択
>> フレーム/アニメーション拡張子から連番の出力名を選択できる(名前.#.拡張子を推奨)
>> **フレームパディング** : ファイル名につく数字の桁数
>> FrameRange
>>> レンダリングするアニメーションのフレーム範囲を設定
>> イメージサイズ
>>> 出力する画像の解像度を設定できる

> **レンダー設定/ArnoldRenderer**
>> Sampling
>>> 各項目ごとに数値を上げれば上げるほどノイズの少ない画像が出力できる(その分レンダリング時間は伸びる)
>>> Camera(AA) : アンチエイリアス
>>> Diffuse : 影の描画
>>> Specular : 反射の描写
>>> SSS : 肌の透け感

> アニメーションをレンダリングする場合は**レンダー/シーケンスレンダー**を使う(左上の項目をレンダリングに変更する)

プレイブラスト
> **ウィンドウ/プレイブラスト**から
>> アニメーターはいちいちレンダリングしていたら重たいため、プレイブラストをつかう
>> **表示/プレイブラストの表示**から表示物を設定できる

ZBrushからオブジェクトをインポートする
> ZBrushのエクスポートから、オブジェクトをABCファイルとして書き出す
> キャッシュ/GPUキャッシュからABCファイルを選択する
>> この状態はメモリ領域にしか保存されないため、内容が消えてしまう
>> そのため、このプロジェクトファイル(.ma)と.abcファイルは残して保存しておく必要がある(同一ディレクトリ下にあることを推奨)
>> この方式であれば、後からZBrushで修正したものを同一.abcに上書きすれば常に最新のモデルでmayaから作業することができる
 
> #注意
>> プラグインマネージャーからabcImportをロード自動ロードをチェックにする
>> gpuCacheをロード自動ロードをチェックにする

キーバインド
> ウィンドウ/プリファレンス/ホットキーエディタから好きなキーバインドを登録できる

プラグイン

>> ツールのアイコンを登録したい際は、C:\Users\amg\Documents\maya\2027\ja_JP\prefs\icons\ の中に.pngを入れる

> 入れたらmaya内の下のウィンドウに、ツール名();を入力する。(.melならMEL,.puthonならputhon表示にする

> アイコンが表示されるが、好きなアイコンに変えたい場合は、アイコンを右クリックし、アイコンの参照パスを、iconディレクトリ内にある.pngを指定する
